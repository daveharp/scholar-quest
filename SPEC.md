# ScholarQuest - Scholarship Finder App

## App Overview

**Name:** ScholarQuest  
**Type:** React Web Application  
**Purpose:** Proactively find and track college scholarships for high school students  
**Target Users:** High school juniors/seniors, parents, counselors

---

## Core Features

### 1. Scholarship Search
- Filter by: Deadline, amount, GPA, residency, field of study
- Save favorites
- Track application status
- Match score based on profile

### 2. Scholarship Details
- Eligibility requirements
- Application steps
- Required documents
- Success tips
- Deadlines with countdowns

### 3. Profile Builder
- GPA, test scores
- Major/interests
- Demographics
- Financial need
- Extracurriculars

### 4. Application Tracker
- Status: Not Started, In Progress, Submitted, Accepted/Rejected
- Due date reminders
- Document checklist

### 5. Success Criteria Library
- What makes applications successful
- Essay examples
- Tips from past recipients

---

## Data Sources

- College Board
- Fastweb
- Sallie Mae
- Federal Student Aid
- State grants

---

## Tech Stack

- React 18 + Vite
- CSS (dark theme)
- LocalStorage for persistence

---

## Run Locally

```bash
cd scholar-quest
npm install
npm run dev
```
